Node.js Server SDKs

FeedHenry platform offers several NPM modules that are freely available as OpenSource. Below are details on our Server-Side SDK modules:

  • fh-amqp-js: client wrapper for communication with a AMQP cluster.
  • fh-db: client wrapper for communication with MongoDB.
  • fh-reportingclient: client library for Node.js applications that allows messages to be sent to a Feedhenry Reporting Server.
  • fh-security: client library for key generation, data encryption/decryption and key storage/retrival.
  • fh-statsc: client library allowing applications to send stats to the stats server.
  • fh-mbaas-client: client module for performing MBaaS operations.
  • fh-mbaas-express: the FeedHenry MBaaS running on top of the famous Express.js
  • fh-mbaas-api: the FeedHenry MBaaS APIs for Node.js cloud apps.

To learn more about Node.js Server-Side SDKs, refer to the API documentation which also comes with code snippets for each feature.
